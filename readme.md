### QMCE (LX)

## What's this?

QMCE 是基于表 Q 内核及部分包装代码的一个 QQ 手表实现，主要使用 Kotlin 开发， UI 使用 Wear Compose 完成。
UI 主要适配圆表，方表没做~~因为太懒~~。

目前项目还在活跃开发中！

## Where to get?

加入 QQ 群 1082439478 ，入群看消息和公告。

## How to build?

很简单 先克隆然后跑`./gradlew assemble`即可
